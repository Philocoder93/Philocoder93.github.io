Schelling Model

This project is a fully functioning version of Thomas Schelling's famous computational model of racial segregation. It was constructed using HTML, CSS, JavaScript, and JQuery.

Smart Search

This project is an app that allows you to create articles, persist them in a database, and search over them for certain keywords. It was created using Ruby on Rails, Devise, and PostgreSQL. It was deployed on Heroku.

Mood Player

This project is an app that allows you to create playlists of songs for certain moods with people all over the world. It was created using a Ruby on Rails backend, an Angular 1 front end, and the Spotify API. The front end was deployed to github pages, the backend was deployed to Heroku.
