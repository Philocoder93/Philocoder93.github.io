Schelling Model

This project is a fully functioning version of Thomas Schelling's famous computational model of racial segregation. It was constructed using HTML, CSS, JavaScript, and AngularJS. It was deployed to GitHub Pages

Smart Search

This project is an app that allows you to create articles, persist them in a database, and search over them for certain keywords. It was created using Ruby on Rails, Devise, and PostgreSQL. It was deployed on Heroku.

Mood Player

This project is an app that allows you to create playlists with people all over the world based on the mood you're in. It was created using Ruby on Rails, AngularJS, and the Spotify API. The front end was deployed to GitHub Pages, the back end was deployed to Heroku.
